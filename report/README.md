# Typst - Report

## Install

```bash
# Ubuntu
snap install typst

# Arch
sudo pacman -S typst
```

## Build

```bash
typst compile relatorio.typ
```

or (if you are in the root of the project):

```bash
typst compile relatorio/relatorio.typ
```

## Watch/Hot Reload

```bash
typst watch relatorio.typ
```

or (if you are in the root of the project):

```bash
typst watch relatorio/relatorio.typ
```
